# 늑대플 개인정보 처리방침

최종 수정일: 2026년 2월 13일

## 1. 수집하는 정보

본 확장 프로그램은 다음 정보를 처리합니다:

- 블로그 에디터에 작성 중인 텍스트 (AI 줄바꿈 및 금칙어 검사 기능 사용 시)
- 무작위 생성된 브라우저 식별자 (인증용, MAC 주소 등 실제 기기 정보가 아님)
- 사용자가 입력한 BLAI API 키

## 2. 정보의 이용 목적 및 전송 대상

| 기능 | 전송 대상 | 전송 데이터 | 목적 |
|------|----------|-----------|------|
| API 인증 | api.blai.co.kr | API 키, 브라우저 ID | 사용자 인증 |
| AI 줄바꿈 | api.blai.co.kr | 블로그 글 텍스트 | 텍스트 줄바꿈 처리 |
| 금칙어 검사 | section.blog.naver.com | 검사 대상 텍스트 | 금칙어/성인어 감지 |

## 3. 정보의 저장

- API 키와 설정값은 Chrome 브라우저 저장소(chrome.storage)에 로컬 저장됩니다.
- 외부 서버에 개인정보를 영구 저장하지 않습니다.
- 전송된 텍스트는 처리 후 즉시 폐기되며, 서버에 보관되지 않습니다.

## 4. 제3자 제공

수집한 정보를 제3자에게 판매하거나 광고 목적으로 사용하지 않습니다.

## 5. 사용자의 권리

- 확장 프로그램 삭제 시 모든 로컬 저장 데이터가 자동 삭제됩니다.
- API 키를 삭제하면 외부 서버 통신이 중단됩니다.
